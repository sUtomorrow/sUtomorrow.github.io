еңЁwin10дёҠжҗӯе»әpython3е’Ңtensorflow-gpuзҺҜеўғ

еҸ‘иЎЁдәҺ 2018-03-10 | еҲҶзұ»дәҺ зҺҜеўғжҗӯе»ә

е®үиЈ…Anaconda2е’ҢAnaconda3

еӣ дёәйңҖиҰҒз”ЁеҲ°дёҖдәӣpython2зҡ„д»Јз ҒпјҢеӣ жӯӨеҶіе®ҡеҗҢж—¶е®үиЈ…Anaconda2е’Ң3пјҢжҗӯе»әpython2е’Ң3е…ұеӯҳзҡ„зҺҜеўғгҖӮ
йҰ–е…Ҳе®үиЈ…Anaconda3дҪңдёәдё»иҰҒзҺҜеўғпјҢе®үиЈ…ж–Үд»¶еҸҜд»ҘеңЁAnacondaе®ҳзҪ‘дёҠдёӢиҪҪпјҢжҲ‘жҠҠе®ғе®үиЈ…еңЁCзӣҳдёӢзҡ„Anacondaзӣ®еҪ•дёӢпјҢдёҖи·ҜдёҠжІЎжңүд»Җд№Ҳзү№еҲ«йңҖиҰҒжіЁж„Ҹзҡ„пјҢдёӨдёӘйҖүйЎ№йғҪйҖүдёӯе°ұеҘҪдәҶпјҢе®үиЈ…е®ҢеҗҺеңЁAnaconda3е®үиЈ…зӣ®еҪ•дёӢзҡ„envsж–Үд»¶еӨ№дёӢж–°е»әpy2зӣ®еҪ•пјҢеҰӮдёӢеӣҫпјҡ

е®үиЈ…еҘҪеҗҺе°ұеҸҜд»ҘжӯЈеёёдҪҝз”Ёpython2е’ҢPython3зҡ„зҺҜеўғдәҶпјҢе№¶дё”еҸҜд»Ҙйқһеёёз®ҖеҚ•зҡ„еҲҮжҚўгҖӮ
еҸӘйңҖеңЁе‘Ҫд»ӨиЎҢиҫ“е…ҘеҰӮдёӢе‘Ҫд»ӨеҚіеҸҜеҲҮжҚўеҲ°python2зҺҜеўғ,иҝҷдёӘpy2е°ұжҳҜдёҠйқўеңЁAnaconda3е®үиЈ…зӣ®еҪ•дёӢзҡ„envsж–Үд»¶еӨ№дёӢзҡ„py2зӣ®еҪ•еҗҚгҖӮ

activate py2

з„¶еҗҺз”Ё

deactivate py2

е°ұеҸҜд»ҘеҲҮжҚўеӣһpython3зҺҜеўғ

е®үиЈ…cuda

еңЁcudaе®ҳзҪ‘йҖүжӢ©дёӢиҪҪзүҲжң¬пјҢжҲ‘йҖүжӢ©cuda9.0зүҲжң¬пјҢжҲ‘зҡ„з”өи„‘йҖӮеҗҲзүҲжң¬еҰӮдёӢпјҡ

з„¶еҗҺејҖе§ӢдёӢиҪҪпјҢ1.4GпјҢйңҖиҰҒдёҖе®ҡж—¶й—ҙгҖӮ
дёӢиҪҪе®ҢжҲҗеҗҺзӣҙжҺҘе®үиЈ…пјҢжҲ‘зӣҙжҺҘдҪҝз”Ёзҡ„зІҫз®Җе®үиЈ…пјҢжӣҙж”№зӣ®еҪ•йҖүеңЁDзӣҳдёӢпјҢз„¶иҖҢе№¶жІЎжңүд»Җд№Ҳж•ҲжһңпјҢиҝҳжҳҜиЈ…еҲ°дәҶCзӣҳпјҢдҪҶжҳҜCзӣҳдёӢйқўиҝҳжҳҜдјҡжңүдёҖдёӘж–Үд»¶еӨ№пјҢеҰӮеӣҫпјҡ

е®үиЈ…е®ҢжҲҗеҗҺиҝҳйңҖиҰҒдёӢиҪҪcudnnпјҢиҝҷдёӘд№ҹеңЁе®ҳзҪ‘еҸҜд»ҘжүҫеҲ°пјҢйңҖиҰҒжіЁеҶҢе№¶еЎ«еҶҷй—®еҚ·жүҚиғҪдёӢиҪҪгҖӮ
дёӢиҪҪзӣёеә”зүҲжң¬пјҢд№ӢеҗҺи§ЈеҺӢеҮәжқҘпјҢеҸ‘зҺ°жңүеҮ дёӘж–Үд»¶еӨ№е’ҢCudaе®үиЈ…зӣ®еҪ•дёӢзҡ„ж–Үд»¶еӨ№еҗҢеҗҚпјҢйӮЈд№ҲзӣҙжҺҘжҠҠж–Үд»¶еӨ№дёӢзҡ„еҶ…е®№жӢ·иҙқеҲ°Cudaе®үиЈ…зӣ®еҪ•дёӯзҡ„еҜ№еә”ж–Үд»¶еӨ№дёӢе°ұеҘҪдәҶгҖӮ
иҝҳиҰҒжіЁж„Ҹзҡ„жҳҜжҠҠдёӢйқўзҡ„и·Ҝеҫ„еҠ е…ҘеҲ°pathзҺҜеўғеҸҳйҮҸдёӯпјҡ

C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v8.0\lib\x64

cudaзҡ„е®үиЈ…еҹәжң¬дёҠе°ұз»“жқҹдәҶгҖӮ
е‘Ҫд»ӨиЎҢдҪҝз”Ё

nvcc --version

еҚіеҸҜжҹҘзңӢcudaзүҲжң¬еҰӮдёӢпјҡ

е®үиЈ…tensorflow-gpu

иҝҷдёӘжҳҜжңҖз®ҖеҚ•зҡ„дёҖжӯҘпјҢеҸӘйңҖиҰҒеңЁе‘Ҫд»ӨиЎҢдёӢдҪҝз”ЁеҰӮдёӢе‘Ҫд»ӨеҚіеҸҜе®ҢжҲҗ:

pip install tensorflow-gpu

зӯүеҫ…еҚіеҸҜ

иҝҷдёӘж—¶еҖҷйҒҮеҲ°дәҶдёҖдёӘbugпјҢиҜҙжҳҜpipе®үиЈ…дёҖдәӣеҢ…зҡ„ж—¶еҖҷжңүзј–з Ғй—®йўҳпјҢgoogleдәҶдёҖдёӢпјҢжҢүз…§иҝҷйҮҢзҡ„еҠһжі•пјҢдҝ®ж”№дәҶpipзҡ„init.pyж–Үд»¶пјҢе®ҢзҫҺи§ЈеҶігҖӮ

жөӢиҜ•зҺҜеўғ

еңЁдәӨдә’жЁЎејҸдёӢпјҢеј•е…Ҙtensorflowз„¶еҗҺеҲӣе»әдёҖдёӘsessionпјҢеҸҜд»ҘеҸ‘зҺ°жүҫеҲ°дәҶдёҖйў—GPUпјҲжҲ‘з”өи„‘дёҠе°ұдёҖдёӘпјүпјҢиҜҙжҳҺзҺҜеўғжҗӯе»әжҲҗеҠҹпјҢжҺҘдёӢжқҘе°ұеҸҜд»ҘејҖе§Ӣж„үеҝ«зҡ„зҺ©иҖҚдәҶгҖӮ

еҺҹеҲӣж–Үз« пјҢиҪ¬иҪҪиҜ·жіЁжҳҺеҮәеӨ„гҖӮ

# tensorflow-gpu # python3 # win10

  • з«ҷзӮ№жҰӮи§Ҳ
